# Prompt Variable Weaver (prompt-variable-weaver)

プロンプト内の変数（空欄）を自動で抽出し、専用の入力フォームから誰でも簡単に高精度なプロンプトを作成できる、完全スタンドアロン（単体動作）型のPWAアプリケーションです。

サーバー不要・管理の手間ゼロで、すべてのデータはあなたのパソコン内だけで安全に完結します。

---

## 📁 構成ファイル
1. **`index.html`** : アプリケーションの本体（画面と処理）
2. **`samples.js`** : 最初から入っている15種類の「内蔵サンプルテンプレート集」
3. **`manifest.json`** : スマホやPCに「アプリ」としてインストールするための設定書
4. **`sw.js`** : ネットが繋がっていなくても一瞬で起動させるためのサービスワーカー
5. **`icon_192.png` / `icon_512.png`** : アプリの顔となるアイコン画像（各自用意）

---

## 💡 知っておくと超便利な「データ管理」の仕組み

このアプリのデータ管理は**「備え付けの教科書」**と**「自分専用のノート」**の2つの組み合わせで動いています。

### ① 内蔵サンプル（引き出しメニュー）
- `samples.js` に入っている15種類の優秀なプロンプトは、アプリを開いた瞬間に**裏側で自動的に一瞬で読み込まれます**。
- 画面左上の「-- テンプレートを選択 --」というドロップダウン（引き出し）を開くだけで、いつでもすぐに使えます。

### ② 「保存」と「読込」ボタン（自分だけのバックアップ機能）
あなたがネット（Chaproなど）から集めて新しく作った「オリジナル・テンプレート」を一生守るための、2つで1つの連動コンボボタンです。

- **【保存】ボタン（外に出す）**
  自分で作ったお気に入りのテンプレートたちが溜まってきたら、このボタンをポチッと押します。すると、それらが丸ごと1つの `.json` というファイルになってパソコンにダウンロード（バックアップ）されます。
- **【読込】ボタン（中に連れて帰る）**
  パソコンの買い替えや、ブラウザの履歴削除などで「自分で作ったデータが消えちゃった！」という時に使います。保管しておいた `.json` ファイルをこのボタンから読み込ませるだけで、一瞬であなたのオリジナルプロンプト集が元通りに完全復元されます。

---

## 🚀 使い方・インストール手順
1. すべてのファイル（5つのコードファイルと2つのアイコン画像）を、**同じ1つのフォルダ**に入れます。
2. ローカルサーバー（VS Codeの拡張機能「Live Server」など）を使って `index.html` をブラウザで開きます。
3. ブラウザのURLバーの右側に表示される「インストール」アイコン（パソコンの画面マークや＋マーク）を押します。
4. デスクトップにアイコンが作成され、次からはクリック一発で独立した「自分専用アプリ」として爆速起動します！
